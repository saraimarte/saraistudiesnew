# Add or Drop Decisions

For these types of questions we are always talking about adding or dropping stores, segments or departments.

We drop a store if we loose more money when keeping the store.

We keep the store if we loose more money when dropping the store.

# Make or Buy Decisions

To decide if we will make or buy something- we focus on the cost of doing so. Whichever decision requires less money is the one we choose.

If it costs less money to buy then we buy it.

If it costs less money to make then we make it.

# Accept or Reject Special Order
