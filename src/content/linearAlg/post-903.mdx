We can use determinants to *determine* if a matrix is invertible or not. 

How do we find determinants?

Well, it depends on the matrix you're working with. 

But note that you can only find determinants of square matricies like (2 x 2), (3 x 3), (4 x 4), (n x n) matricies, etc.
