# Find Length of a Vector 

<Rulebox>
If you have a vector **v**, then the length of the vector is denoted by ||**v**||.

||**v**|| is calculated simply by taking all the entries in the vector, squaring them, adding them together, and taking the square root of the sum like so,

<Latex formula=" \mathbf{v} = \begin{bmatrix} 1 \\ 2 \\ 3 \end{bmatrix}, \quad ||\mathbf{v}|| = \sqrt{1^2 + 2^2 + 3^2} = \sqrt{14}" />
</Rulebox>

# Find Distance Between 2 Vectors 

<Rulebox>
If you have vectors **u** and **v**, then the distance between them is simply the length of the vector that connects them that is **u**-**v**, that is, ||**u**-**v**||. 
</Rulebox>
