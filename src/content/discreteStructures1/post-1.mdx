# What is a Proposition?

The first thing you learn in discrete is what a proposition is. 

<RuleBox>

A **proposition** is a declarative statement that can either be true or false, not both. It cannot be a question or contain undefined variables. 

</RuleBox>

### Types of Implications (CIC 'Sick'/SOB)

<RuleBox>
The default implication is p → q. This translates to "If p is true then q must be true."

We can have variations of these implications. We call these variations the converse, inverse and contrapositive. You may be asked to find the converse of an implication but instead of memorizing you can remember the phrase, "If you're **CIC** (Sick), you're **SOB**bing.", where 'sick' is spelt 'cic'.
