G = (V, Σ, S, P) is a grammar where:  

**V (alphabet):** is a set of variables (aka the alphabet; it can include both terminal and non-terminal variables)         

**Σ (or T):** contains only terminals variables (your strings should only contain these variables at the end)  

**S:** is the start variable (the one you will use to start the derivation process)    

**P:** is a set of rules (productions) that describe how to form strings using the grammar   

Note: Variables can also be called symbols or characters.


</RuleBox>

<RuleBox align = "left">

<CustomThumb
  iframeSrc="https://www.youtube.com/embed/PK7kcAkQjB8?si=As0EAys2KBh1wSfv&start=5"
  src={img1.src}
/>

M = (Q, Σ, δ, q₀, F) is a Finite State Machine where:  

**Q:** is a set of all states          

**Σ (or T):** is the set of all input symbols (all the symbols that your strings are made up of) 

**δ (delta):** is the transition function. **DFA:** Q x Σ → Q **NFA:** Q x Σ → P(Q)

**q₀:** is the starting state

**F:** is a set of all final/accepting states (these are double circled in state diagrams)

Note: Symbols can also be called variables or characters.

</RuleBox>

<Section t = "Turing Machines">

<RuleBox align = "left">

<CustomThumb
  iframeSrc="https://www.youtube.com/embed/PK7kcAkQjB8?si=As0EAys2KBh1wSfv&start=5"
  src={imgTM0.src}
/>

M = (Q, Σ, Γ, f, q₀, F) is a Turing Machine where:  

**Q:** is a set of all states          

**Σ (T/sigma/input alphabet):** is the set of all input symbols (all the symbols that your strings are made up of) 

**Γ (gamma/tape alphabet):** is the set of all tape symbols (all the symbols that you can ever see on the tape; it includes Σ) 

**f (delta):** is the transition function. **f:** Q x Γ → Q x Γ x M (where M contains the elements L and R where L = left R = right)

**q₀:** is the starting state

**F:** is a set of all halting states (states with no transitions)

Note: We adopt the rule that if a state has no transition fro a given input, then the TM halts.

</RuleBox>
