# 10.2 Counting by Using the Fundamental Principle of Counting (Product/Multiplication Rule)

<RuleBox align ="left">
**Def.** The **Fundamental Principle of Counting**, also known as the multiplication rule or the product rule, says if you have multiple sequential events, or actions/tasks, and they each have a specific number of ways to occur, you can multiply the number of ways for each event to find how many ways all the events could happen together.

E.g. If Event A can occur in n ways and Event B can occur in m ways, then the two events can occur together in n x m ways.

</RuleBox>

## 10.2.1 Counting by Using the Factorial Theorem

The factorial theorem is just a shortcut for a specific type of problem you solve using the fundamental principal of counting. 

<RuleBox align ="left">
**Def.** The **Factorial Theorem of Counting** says that the total number of different ways to arrange n distinct objects is n!
</RuleBox>
