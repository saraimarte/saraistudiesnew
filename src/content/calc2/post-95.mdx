# 10.8 Taylor Series 

A cool thing we can do is turn any function into a polynomial.

Why would we want to do that?

Because polynomials are easier to take derivatives of and integrate!

So if we can turn any scary looking function into a regular polynomial then why not!

Below, you can see that a function f(x) is literally equal to a series. We call this a Taylor Series. 

<RuleBox>
<Latex formula="f(x) = \sum_{k=0}^\infty \frac{f^{(k)}(a)}{k!}(x - a)^k" /> 
</RuleBox>

If you write the series in expanded form, you will actually get a polynomial. 

We call this the Taylor Polynomial because it was generated by the Taylor Series.

When do we know when to stop expanding/plugging in? 

Well, they will tell you to find the Taylor Polynomial for a given function at a given order. That order is n. n for the number of times you will plug into the series.  

So, we denote that as T<sub>n</sub>...

<RuleBox>
<Latex formula="T_n(x) = f(a) + f'(a)(x - a) + \frac{f''(a)}{2!}(x - a)^2 + \cdots + \frac{f^{(n)}(a)}{n!}(x - a)^n" />
</RuleBox>

They will ask you to find the Taylor Polynomials for a given function at a given order and center.

<Problem n = "1">
<Latex formula="
f(x) = (1+x)^{\frac{2}{3}}, \quad T_3(x) \text{ at } a = 0
"/>
</Problem>

<Solution>

T<sub>3</sub>(x) = 1 + 2/3x - (2/9)/2!x<sup>2</sup> + (8/27)/3! x<sup>3</sup>

</Solution>
