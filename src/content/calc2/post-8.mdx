# Products of Powers of Sines and  Cosines

<RuleBox>

<Latex formula = "\int \sin^m(x) \cos^n(x) \, dx" />

<p>With these types of problems you have 3 scenerios.</p>

<p><b>Case 1 </b>If<b> m odd</b> → Rewrite using <b>sin<sup>2</sup>(x) = 1 -cos<sup>2</sup>(x)</b> then do u-Sub</p>

<p><b>Case 2 </b>If<b> n odd </b> → Rewrite Using <b>cos<sup>2</sup>(x) = 1 -sin<sup>2</sup>(x)</b> then do u-Sub</p>

<p><b>Case 3 </b>If <b> n & m both even </b> → we rewrite using either of the following</p> 

<Latex formula = "\mathbf{\textcolor{red}{\boldsymbol{\sin}^2(x)}} = \frac{1 \mathbf{\textcolor{red}{\boldsymbol{-}}} \cos(2x)}{2}" />
<Latex formula = "\cos^2(x) = \frac{1 + \cos(2x)}{2}" />

<b> Tip! </b>
<p> The formulas above are the same thing the only thing that is different are the signs. I remember sin<sup>2</sup>(x) as having the negative sign because <span style = "color:red; font-weight:bold"> sin is bad </span> so <span style = "color:red; font-weight:bold">sin means negative. </span></p>

<p>These are called **half** angle formulas which makes sence because they both **halve** the 1 +/- cos(2x).</p>

</RuleBox>

# Integrating Square Roots

<RuleBox>

We can use the following identity to eliminate a square root:

<Latex formula = "\cos^2(\theta) = \frac{1 + \cos(2\theta)}{2} \, \text{ or } \, 1 + \cos(2\theta) = 2\cos^2(\theta)"/>
<Latex formula = "\sin^2(\theta) = \frac{1 - \cos(2\theta)}{2} \, \text{ or } \, 1 - \cos(2\theta) = 2\sin^2(\theta)"/>

These are the same as the first brainbox- they're the half angle identities. 

</RuleBox>

# Products of Powers of Tangents and Secants. 

<RuleBox>

<Latex formula = "\int \tan^m(x) \sec^n(x) \, dx" />

<p>With these types of problems you have 3 scenerios.</p>

<p><b>Case 1 </b>If<b> m odd</b> → Rewrite using <b>tan<sup>2</sup>(x) = sec<sup>2</sup>(x) -1</b> then do u-Sub</p>

<p><b>Case 2 </b>If<b> n even </b> → Rewrite Using <b>sec<sup>2</sup>(x) = 1 + tan<sup>2</sup>(x)</b> then do u-Sub</p>

<p><b>Case 3 </b>If<b> m even and n = 0 </b> → Rewrite Using <b>tan<sup>2</sup>(x) = sec<sup>2</sup>(x) -1</b> then do u-Sub</p>

Afterwards, you may do integration by parts if needed.

<p><b>Tip!</b> The way I remember these rules are mnm like the choclate m&m's and then tan sec tan=.</p>
</RuleBox>

# Products of Sines and  Cosines

<RuleBox>

<p>These we can all solve with integration by parts but there are identities you can simply memorize to solve these faster.</p> 

<p>If you have the following integrands you can use rewrite them using their respective identities. </p>

<Latex formula = "\sin(mx) \sin(nx) = \frac{1}{2}[\cos((m-n)x) - \cos((m+n)x)]"/>

<Latex formula = "\sin(mx) \sin(nx) = \frac{1}{2}[\sin((m-n)x) + \sin((m+n)x)]"/>

<Latex formula = "\cos(mx) \cos(nx) = \frac{1}{2}[\cos((m-n)x) + \cos((m+n)x)]"/>

</RuleBox>
