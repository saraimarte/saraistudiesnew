**2.** For a time series with 10 observations, we can have 9 averages in moving averages with duration L = 3. True or False?

<Solution>

**Answer: False.**

Number of averages = N - (L - 1)

If N = 10 and L = 3 then

10 - (3 - 1) = 8

So, the number of averages is 8 not 9.

</Solution>

# Fitting and Forecasting

Just like we had the linear, quadratic and exponential models...we now have the linear, quadratic, exponential _trend_ models.

They are the exact same thing except this time the years are recoded. As in, if we have years 2000, 2001 and 2002 then we will _recode_ those years to be 0, 1 and 2 and use those code names as our x-values instead of the given times.

Why do we do this?

Because if we use the regular x values (the years) we will get the wrong summary output and hence the wrong equation/model.

# Autoregressive Model

We use this model for time series data that is autocorrelated- meaning it's data points are highly correlated or influenced by previous data points.

We have three models to choose from.

(1) **First-Order Model:** Å¶<sub>i</sub> = A<sub>0</sub> + A<sub>1</sub>(Å¶<sub>i</sub>-1)  
(2) **Second-Order Model:** Å¶<sub>i</sub> = A<sub>0</sub> + A<sub>1</sub>(Å¶<sub>i</sub>-1) + A<sub>2</sub>(Å¶<sub>i</sub>-2)  
(3) **Third-Order Model:** Å¶<sub>i</sub> = A<sub>0</sub> + A<sub>1</sub>(Å¶<sub>i</sub>-1) + A<sub>2</sub>(Å¶<sub>i</sub>-2) + A<sub>3</sub>(Å¶<sub>i</sub>-3)

(1) **First-Order Model:** A data point is highly influenced by it's previous data point (1st before it).  
(2) **Second-Order Model:** A data point is highly influenced by it's 2nd previous data point (2nd before it).  
(3) **Third-Order Model:** A data point is highly influenced by it's 3rd previous data point (3rd before it).

Here, A<sub>0</sub>: intercept. (Å¶<sub>i</sub>-1): x-value.

We choose the model based on the better adj.R<sup>2</sup> and p-value.
