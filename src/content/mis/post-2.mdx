# Installing MySQL (For Windows)

**1.** Go <a href = "https://dev.mysql.com/downloads/installer/">here</a>   
**2.** Choose 8.0.39 as the version.  
**3.** Download the biggest file size (Windows (x86, 32-bit), MSI Installer Size: 303.6M)  
**4.** Go to downloads, click on the file and it should run the installer.   
**5.** In the installer click Server Only  
**6.** Click next and execute. Keep clicking on next until you get to Type and Networking. Here, for the config type choose development computer.   
**7.** Click next until it asks you for a password. Make sure to save it!

# Installing MySQL Workbench (For Windows)

**1.** Go <a href = "https://dev.mysql.com/downloads/workbench/">here</a>.  
**2.** Download "Windows (x86, 64-bit), MSI Installer 41.7M."
