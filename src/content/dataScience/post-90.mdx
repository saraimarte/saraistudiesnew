# PCA 

<RuleBox>
**Def.** **Principle Component Analysis (PCA)** is a dimensionality reduction technique that simplifies complex datasets while retaining as much variability (information) as possible.
</RuleBox>

# PCA Implementation Methods

There are two main ways (there's also the sequential algorithm) you can implement PCA: 

**1. Singular Value Decomposition (SVD):** decomposes a matrix into three distinct matricies U, Σ, V<sup>T</sup>, where the column vectors in V (or rows in V<sup>T</sup>) are the principal components of the data.   

**2. Eigen Decomposition of the Covariance Matrix:** involves finding the covariance matrix C of the data and then decomposing that matrix into three distinct matricies W, λ , W<sup>-1</sup>,  where the columns in W are the principal vector components.
