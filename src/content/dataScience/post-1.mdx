# What is Pandas 

Pandas is a data analysis library in Python. 

It allows you to store data in it's data structures. 

Each data structure has attributes and methods you can use to clean, explore and analyze data. 

It also allows you to work with data files you already have like excel, csv, json, html, xml, txt, files etc. 

# Key Data Structures

There are two main data structures in pandas: (a) DataFrame and (b) Series.

## Creating data frames

<Code lang="python" fileName = "FileName.ipynb" code={
