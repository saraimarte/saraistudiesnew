Data types are types of...well data that we can perform operations on.

There are 5 types of data we will deal with in Java.

<li style="list-style:none"> (1) String (Text: 'hello world')</li>
<li style="list-style:none"> (2) int (Integers: 1, 4, -2, -20) </li>
<li style="list-style:none"> (3) double (Decimals: 2.0 2.44) </li>
<li style="list-style:none"> (4) boolean (True or False) </li>
<li style="list-style:none"> (5) char (Characters: & or 'B') </li>

## Casting

Casting is all about changing a _numerical_ data type to another (so it doesn't work with strings because a string is not numerical).

### Example 1 | Integer to a Double

<Code lang="java" fileName="CastingIntegerToDouble" code={`int num1 = 20;

double num2 = (double) num1;

System.out.println(num2);

// 20.0`} />

### Example 2 | Double to an Integer

<Code lang="java" fileName="CastingDoubleToInteger" code={`double weight = 140.20;

// here the (int) turns the weight which is a double into an integer
int weightAsAnInteger = (int) weight;

System.out.println(weightAsAnInteger);

// 140
// Notice how it rounded down.
// (int) will always round down even if the double is 140.90`} />

But what if we do want to turn a string into an integer?

We can use what are called the system's version of casting or the system's method which uses parsing.

### Example 3 | String to an Integer

<Code lang="java" fileName="StringToInteger" code={`String ageAsString = "103"; //String

int ageAsInteger = Integer.parseInt(ageAsString); // Turning String into Integer

System.out.println(ageAsInteger);

//103`} />
