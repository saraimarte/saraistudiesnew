Alrighty, before we start with real code we have to do pseudo code!

Pseudo code is just a way to tell the computer what to do in English instead of Java.

Let's practice!

### Example 1 | Take in two numbers from the user. Output the second number.

<Code lang="java" fileName="Example 1" code={`READ num1\nREAD num2\nDISPLAY num2`} />

### Example 2 | Set a variable age equal to your age. Add 2 years to it and then subtract 1 year. Display age.

<Code lang="java" fileName="Example 2" code={`SET age TO 20.\nADD 2 TO age\nSUBTRACT 1 FROM age\nDISPLAY age`} />

### Example 3 | Compute an expression so that a variable x is equal to (5 times 2) plus 10

<Code lang="java" fileName="Example 3" code={`COMPUTE x AS 5*2 + 10`} />

### Example 4 | Take two numbers as inputs, add them together and display the sum.

<Code lang="java" fileName="Example 4" code={`READ num1\nREAD num2\nCOMPUTE sum AS num1 + num2\nDISPLAY sum`} />

Note: You can't use `SET sum TO num1 + num2` because SET can only be assigned a variable or a constant, NOT an expression.
You can do something like `SET grade TO 5` because 5 is a constant, or you can do

<Code lang="java" fileName="Example 4" code={`SET otherVariable TO 5\nSET grade TO otherVariable`} />

### Example 5 | Display the remainder you get when you divide 10 by 5.

<Code lang="java" fileName="Example 5" code={`COMPUTE remainder AS 10 % 5\nDISPLAY remainder\n// this should display 0 because 10 divided by 5 is 2 with remainder 0`} />

### Example 6 | Create a program that displays if a number is even or odd.

<Code lang="java" fileName="Example 6" code={`READ number\nIF number % 2 IS 0 THEN\nDISPLAY even\nELSE\nDISPLAY odd\nENDIF`} />

### Example 7 | Write a program to calculate the area of a rectangle. Remember to account for erroneous inputs!

<Code lang="java" fileName="Example 7" code={`READ base\nREAD height\nIF base < 0 OR height < 0 THEN\nDISPLAY ERROR\nELSE\nCOMPUTE area AS base * height\nDISPLAY area\nENDIF`} />

### Example 8 | Write a program to convert a given temperature in Celsius to Fahrenheit. Remember to account for erroneous inputs!

Formula: `celsius * (9/5) + 32`.

Note: The absolute minimum temperature that can exist in Celsius is −273.15 degrees. Anything lower than that is impossible... theoretically.

<Code lang="java" fileName="Example 8" code={`READ degreesInCelsius\nIF degreesInCelsius < −273.15 THEN\nDISPLAY ERROR\nELSE\nCOMPUTE degreesInFahrenheit AS degreesInCelsius * (9/5) + 32\nDISPLAY degreesInFahrenheit\nENDIF`} />

### Example 9 | Write an algorithm that displays only the last digit of a four-digit number given by the user.

<Code lang="java" fileName="Example 9" code={`READ number\nCOMPUTE lastDigit AS number % 10`} />

### Example 10 | Challenge

In response to a gas shortage crisis, a local government has implemented a gas rationing system based on the last digit of vehicles' license plate numbers. Create a program that informs drivers whether they can refuel on even or odd-numbered days depending on the last digit of their license plate number.
If their license plate numbers end with an even digit, they can only get gas on even-numbered days. Otherwise, they can get gas on odd-numbered days.

Assumptions:

- All license plates are four digits long.
- License plates contain only numerical digits.

<Code lang="java" fileName="Example 10" code={`READ licensePlateNumber\nCOMPUTE lastDigit AS licensePlateNumber % 10\nIF lastDigit % 2 IS 0 THEN\nDISPLAY You can get gas on even-numbered days.\nELSE\nDISPLAY You can get gas on odd-numbered days.\nENDIF`} />
